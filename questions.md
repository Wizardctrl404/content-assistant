# Master Question Bank
> Used by /rapidfire and other skills. Claude selects intelligently based on USER_PROFILE.md — industry, gaps, session history, and what's already been answered.

## Selection Rules
- Never ask a question the user has already answered in a previous session (check data files).
- Prioritize questions in categories where USER_PROFILE.md shows gaps.
- For technical/niche users, lean toward contrarian and industry questions.
- For early-stage or unclear-on-identity users, lean toward origin and identity questions.
- Always mix categories — don't ask 5 origin questions back to back.
- Flag any answer that could become a content pillar, signature story, or quotable framework.
